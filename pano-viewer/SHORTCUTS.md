# ⌨️ 快捷键参考卡

## 📋 快速参考

### 文件操作 📁
| 快捷键 | 功能 |
|--------|------|
| `Ctrl + S` | 💾 保存项目 |
| `Ctrl + Shift + S` | 📷 截取当前视角 |

### 编辑操作 ✏️
| 快捷键 | 功能 |
|--------|------|
| `Ctrl + Z` | ↩️ 撤销 |
| `Ctrl + Y` | ↪️ 重做 |
| `Ctrl + Shift + Z` | ↪️ 重做（备选） |
| `Ctrl + C` | 📋 复制场景 |
| `Ctrl + V` | 📄 粘贴场景 |
| `Delete` | 🗑️ 删除当前场景 |

### 热点操作 📍
| 快捷键 | 功能 |
|--------|------|
| `Ctrl + H` | 📍 添加热点 |
| `Ctrl + T` | 📝 添加文字热点 |
| `Ctrl + M` | 🎵 添加背景音乐 |

### 视图控制 👁️
| 快捷键 | 功能 |
|--------|------|
| `F11` | ⛶ 全屏模式 |
| `ESC` | ❌ 退出/关闭 |
| `←` | ⬅️ 上一个场景 |
| `→` | ➡️ 下一个场景 |

---

## 🎯 使用技巧

### 快速工作流
```
1. Ctrl + H     → 添加热点
2. Ctrl + T     → 添加说明文字
3. →            → 切换到下一场景
4. Ctrl + C/V   → 复制配置到其他场景
5. Ctrl + S     → 保存工作
```

### 常见场景

#### 场景编辑
```
1. 选择场景
2. Ctrl + H 添加热点
3. Ctrl + Z 撤销错误操作
4. Ctrl + S 保存
```

#### 批量处理
```
1. Ctrl + C 复制模板场景
2. → 切换到下一场景
3. Ctrl + V 粘贴
4. 重复2-3步
```

#### 预览演示
```
1. 按预览按钮（或自定义快捷键）
2. → 或 ← 切换场景
3. F11 全屏展示
4. ESC 退出
```

## 💡 注意事项

### 快捷键不生效？
✓ 确保焦点不在输入框中
✓ 某些浏览器可能拦截部分快捷键
✓ Mac使用Cmd代替Ctrl

### 最佳实践
✓ 经常使用Ctrl+S保存
✓ 善用Ctrl+Z避免误操作
✓ 使用箭头键快速浏览场景
✓ 批量操作时使用复制粘贴
